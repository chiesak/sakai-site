# 読み物の更新方法

## 毎週の更新はこれだけ

`data/column.json` を開いて、**先頭**に1ブロック追加する。

---

## テンプレート（コピーして使う）

```json
  {
    "id": "20260221a",
    "date": "2026-02-21",
    "title": "ここにタイトルを書く",
    "tags": ["国際社会学1", "授業補足"],
    "body": "本文の1段落目をここに書く。\n改行したいときは\\nを入れる。\n\n空行（\\n\\n）で段落が変わる。\n\n### 小見出し\n\n見出し以降の本文。"
  },
```

**追加する場所：**
```
[            ← ファイルの先頭
  {          ← ここに新しい記事を追加
    "id": ...
  },
  {          ← 既存の記事
    "id": "20260214a",
```

---

## 各項目の説明

| 項目 | 内容 | 例 |
|---|---|---|
| `id` | 日付＋連番（重複しなければ何でもOK） | `"20260221a"` |
| `date` | 公開日（年-月-日） | `"2026-02-21"` |
| `title` | タイトル | `"第4回補足：〇〇について"` |
| `tags` | タグ（下の一覧から選ぶ） | `["国際社会学1", "授業補足"]` |
| `body` | 本文 | 下記参照 |

---

## 使えるタグ一覧

**授業（春学期）**
- `"国際社会学1"`
- `"基礎研究1"`
- `"専門演習1"`
- `"卒業研究1"`

**研究・コラム**
- `"研究コラム"`
- `"移民と博物館"`
- `"学会"`

**その他**
- `"授業補足"`（科目を問わない補足全般）

タグは複数つけられる：`["国際社会学1", "授業補足"]`

---

## 本文の書き方

### 段落を分ける
空行（`\n\n`）を入れると段落が変わる。

```
"body": "1段落目の文章。\n\n2段落目の文章。"
```

### 小見出しをつける
`### 見出し` で小見出しになる。

```
"body": "導入文。\n\n### ここが小見出し\n\n見出し以降の文章。"
```

### 太字
`**太字にしたい部分**` で太字になる。

---

## GitHubへのアップロード

1. `data/column.json` を編集して保存
2. GitHub の `sakai-site` リポジトリを開く
3. `data/column.json` をクリック → 右上の鉛筆アイコン（Edit）
4. 内容を貼り付けて「Commit changes」
5. 1〜2分後にサイトに反映される

---

## よくあるミス

- カンマを忘れる：記事と記事の間は `,` が必要
- 最後の記事にカンマをつける：最後の `}` の後にカンマは**不要**
- 日本語の引用符を使う：`"` は半角の `"` を使うこと

JSONが壊れたかどうかは https://jsonlint.com/ に貼り付けて確認できる。
